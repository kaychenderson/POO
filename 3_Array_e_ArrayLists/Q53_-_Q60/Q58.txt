Q58. [FADE- UFPE- 2023- UFPE- Analista de Tecnologia da Informação- Área: Sistemas] Considere o seguinte código em Java:

public class Main {
  public static void main(String[] args) {
    int[] A = {12, 25, 16, 11, 9, 31, 23, 27, 22};
    System.out.println(A[A[2] - A[4]] - A[6]);
  }
}

A execução desse código resulta na impressão do seguinte valor:
 (X) 4
 ( ) 3
 ( ) 2
 ( ) 1
 ( ) 0

 Passo a passo:
 A[2] - A[4] = (16 - 9) = 7
 A[7] - A[6] = (27 - 23) = 4
